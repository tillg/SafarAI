# SafarAI

A Safari Web Extension that makes the browser AIntelligent.

# Features

- Summarize content of open page
- Ask questions about content of open page
- Discuss / question content of the open page while accessing other resources, i.e. mcp servers, browsing the internet, asking questions to other LLMs...
- Ability to choose the backing LLM: 
  - OpenAI is first, and default (bring your own API key)
  - OpenAI-based API, from other LLMs
  - Local LLM of macOS
